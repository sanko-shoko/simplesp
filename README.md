# simplesp

simple library for computer vision


## compile
code
```
#include "simplesp.h"
using namespace sp;
     
int main(){
    printf("compile test\n");
    return 0;
}
```

command
```
$ cd simplesp
$ g++ compiletest.cpp -I ./ -std=c++11
$ ./a.exe
```
